# UrgencyBridge | Digital Doorbell

A lightweight, web-only emergency communication tool that bypasses social media apps using prioritized emails and real-time chat.

## Features
- **Unique ID Mapping**: Reach someone by their ID without knowing their email.
- **Priority Alerts**: Triggers an urgent notification via Email.
- **Real-time Bridge**: Instant, private browser-to-browser chat.
- **Emergency Dark Mode**: High-contrast UI for stressful situations.

## Setup
1. **Supabase**: Run the schema provided in the documentation to set up the database.
2. **Credentials**: Hardware your Supabase and EmailJS keys into `app.js`.
3. **Deploy**: Follow the instructions below to run online.

## Deployment Instructions

### Vercel (Recommended)
1. Push this code to a **GitHub** repository.
2. Link your GitHub account to [Vercel](https://vercel.com).
3. Select this repository and click **Deploy**. Vercel will provide you with a live URL within seconds.

### GitHub Pages
1. In your GitHub repository, go to **Settings > Pages**.
2. Under "Build and deployment", set the source to **Deploy from a branch**.
3. Select your `main` branch and `/root` folder, then click **Save**.
